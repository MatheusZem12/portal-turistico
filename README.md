# 🌴 Brasil Encantado — Portal Turístico Interativo

## Descrição

Portal turístico interativo desenvolvido para a agência fictícia **Brasil Encantado**, com foco na divulgação de destinos turísticos brasileiros, gastronomia e cultura. O site apresenta informações sobre os principais destinos do país, pratos típicos regionais e formas de contato.

Desenvolvido com **HTML5 semântico** e **CSS externo**, utilizando Flexbox para layout, design responsivo com media queries e efeitos visuais com hover e animações CSS.

## Páginas

- **index.html** — Página inicial com destaques, vídeos e curiosidades
- **destinos.html** — Principais destinos turísticos do Brasil
- **gastronomia.html** — Pratos típicos e cultura gastronômica brasileira
- **contato.html** — Formulário de contato e informações da agência

## Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (estilização externa)
- Flexbox (layout)
- Media Queries (responsividade)
- Google Fonts (tipografia)
